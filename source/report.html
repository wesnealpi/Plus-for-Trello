Special orders:

R (non-zero)
Special 'Remaining' report. Shows only rows with non-zero R.
This order makes sense with filter 'since: All dates' and group by card,list,board,team or user.

To sort by R without this special behavior, order by something else, query, then click the R column header.

To find NOT containing, start with !
Partially supports AND/OR:
When using OR, no term can be negated: label1 OR label2 OR label3
When using AND, all terms must be negated: !label1 AND !label2 AND !label3
If you need a more complex filter, export the report results to a spreadsheet using 'Copy'.

Labels sync note: Plus automatic sync does not always detect all card labeling changes that happened within the last two minutes before sync, or label names on boards where the only change was a renamed label.
To force fresh card label results in those cases click on sync from the Chrome Plus menu or the top weekly report in Trello.
